# 安装

## 1. 安装最新版的 conky

去这里看安装方法 https://github.com/brndnmtthws/conky/blob/master/README.md#quickstart 

## 2. 拷贝文件到对应的目录

```bash
    cp .conkyrc ~/.conkyrc
    cp -rf .conkycolors ~/.conkycolors
    cp -rf .local/share/fonts/conkycolors ~/.local/share/fonts/conkycolors
```
